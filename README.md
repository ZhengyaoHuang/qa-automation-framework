# QA Automation Framework

A basic QA automation framework built with Selenium and PyTest.

## Tech Stack
- Python
- Selenium WebDriver
- PyTest
- Requests
- Page Object Model (POM)

## Project Structure
- `pages/` page objects
- `tests/` test cases
- `utils/` driver setup
- `conftest.py` pytest fixtures

## Features
- UI login automation test
- API mock test
- Page Object Model structure
- Reusable driver fixture

## How to Run
```bash
pip install -r requirements.txt
pytest -v
